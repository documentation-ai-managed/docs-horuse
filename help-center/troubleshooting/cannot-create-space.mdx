---
title: Cannot Create New Space
description: Resolve issues creating a new Space in Horuse.
---

## Symptoms

- "Create Space" button is disabled
- Error: "Unable to create Space"

## Causes

- User limit reached
- Missing required fields
- Network issues

## Solution
<Steps>
  <Step title="Check your plan limits">
    Free plans allow a limited number of Spaces.
  </Step>
  <Step title="Complete all required fields">
    Make sure the Space name and description are filled in.
  </Step>
  <Step title="Retry after checking network connection">
    Reload and try again.
  </Step>
</Steps>
