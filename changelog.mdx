## How to Update

Update Horuse to the latest version in minutes. Follow these steps for a smooth upgrade.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command.
    
````bash
npm list horuse
````
  </Step>
  <Step title="Install Latest" icon="download">
    Use your package manager.
    
    <CodeGroup tabs="npm,yarn">
````bash
npm install horuse@latest
````
````bash
yarn add horuse@latest
````
    </CodeGroup>
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your development server to apply changes.
    
````bash
npm run dev
````
  </Step>
</Steps>

<Callout kind="tip">
  Always back up your `docs/` folder before updating. Review migration notes below for version-specific changes.
</Callout>

## Migration Notes

For major releases, you might need minor adjustments.

<Expandable title="v1.2.0 Migration" default-open="true">
  Update any `{variable}` references to use backticks in prose: `Use `{API_KEY}` for auth`. No other changes required.
</Expandable>

<Expandable title="v1.1.0 Migration">
  Replace single quotes in JSX attributes with double quotes: `title="Text"` instead of `title='Text'`.
</Expandable>
