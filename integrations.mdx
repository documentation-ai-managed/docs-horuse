## Webhook Setup

Set up webhooks to receive real-time events from Horuse, such as doc updates or page views.

<Steps>
  <Step title="Navigate to Integrations" icon="settings">
    Go to your Horuse dashboard at `https://dashboard.example.com` > Settings > Integrations > Webhooks.
  </Step>
  <Step title="Add Webhook URL" icon="plus">
    Enter your endpoint URL, like `https://your-webhook-url.com/horuse`.
  </Step>
  <Step title="Select Events" icon="bell">
    Choose events: `doc.updated`, `page.viewed`, `user.invited`.
  </Step>
  <Step title="Verify and Save" icon="check-circle">
    Test the webhook and save. Horuse sends a verification payload.
  </Step>
</Steps>

Example webhook payload:

<CodeGroup tabs="JSON">
  ```json
  {
    "event": "doc.updated",
    "doc_id": "doc_123abc",
    "timestamp": "2024-10-15T12:00:00Z",
    "user_id": "user_456def"
  }
  ```
</CodeGroup>

## API Access for Custom Integrations

Use the Horuse API to create, update, and query documentation programmatically.

<Request tabs="JavaScript,cURL">
  ````javascript
  const response = await fetch('https://api.example.com/v1/docs', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'New Doc',
      content: 'Initial content here.'
    })
  });
  ````
  ````bash
  curl -X POST https://api.example.com/v1/docs \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "title": "New Doc",
      "content": "Initial content here."
    }'
  ````
</Request>

<Response tabs="200">
  ```json
  {
    "id": "doc_123abc",
    "title": "New Doc",
    "status": "success"
  }
  ```
</Response>

<ParamField path="title" param-type="string" required="true">
  Document title, max 255 characters.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained from dashboard.
</ParamField>

<Tabs>
  <Tab title="JavaScript SDK" icon="code">
    Install via npm:

    ````bash
    npm install @horuse/sdk
    ````

    Initialize:

    ````javascript
    import { Horuse } from '@horuse/sdk';

    const horuse = new Horuse({ apiKey: 'YOUR_API_KEY' });
    const doc = await horuse.docs.create({ title: 'My Doc' });
    ````
  </Tab>
  <Tab title="Python" icon="python">
    Install via pip:

    ````bash
    pip install horuse-py
    ````

    Usage:

    ````python
    from horuse import HoruseClient

    client = HoruseClient(api_key='YOUR_API_KEY')
    doc = client.docs.create(title='My Doc')
    ````
  </Tab>
</Tabs>

<Callout kind="tip">
  Generate your API key from `https://dashboard.example.com/api-keys`. Store it securely.
</Callout>

## Embedding External Content

Embed iframes, videos, or widgets from external services directly into Horuse docs.

Use this syntax in your doc content:

````markdown
```iframe
https://example.com/widget
height=400
```
````

<Expandable title="Advanced Embedding Options" default-open="false">
  Customize with parameters:

  | Parameter | Description | Example |
  |-----------|-------------|---------|
  | `height`  | Frame height in pixels | `height=500` |
  | `width`   | Frame width (default: 100%) | `width=800` |
  | `sandbox` | Security restrictions | `sandbox=allow-scripts` |
</Expandable>
