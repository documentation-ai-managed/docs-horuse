## Overview

Horuse provides a centralized space to organize, manage, and collaborate on your project documentation. You create structured docs, embed code snippets, and share knowledge with your team effortlessly. Whether you document APIs, user guides, or internal wikis, Horuse scales with your needs.

This platform supports real-time editing, version history, and search across all your content. Start by creating a workspace, then add pages, sections, and embeds to build comprehensive documentation.

<Callout kind="info">
  New to Horuse? Follow the quick start guide to set up your first workspace in under 5 minutes.
</Callout>

## Navigation and Interface Tour

Horuse features an intuitive interface. Follow these steps to explore:

<Steps>
  <Step title="Access Dashboard" icon="monitor">
    Log in at `https://dashboard.horuse.com`. You see your workspaces on the left sidebar.
  </Step>
  <Step title="Create Workspace" icon="plus">
    Click <kbd>New Workspace</kbd>. Name it and select a template.
  </Step>
  <Step title="Add Pages" icon="file-plus">
    Inside a workspace, use the <kbd>+ Page</kbd> button. Organize with drag-and-drop.
  </Step>
  <Step title="Customize Theme" icon="palette">
    Go to Settings > Theme. Apply your brand color like `#3B82F6`.
  </Step>
</Steps>

<Tabs>
  <Tab title="Teams" icon="users">
    Use the Members tab to invite collaborators and set permissions.
  </Tab>
  <Tab title="Solo" icon="user">
    Focus on the Editor view for quick page creation without distractions.
  </Tab>
</Tabs>
